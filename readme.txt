A WordPress plugin to implement a tracking code for Matomo analytics.

The plugin is designed to be light, fast, basic and pretty optionless. It's a set and forget kind of solution.

This is our fork of the 'Tracking code for Matomo, by Sergio Santos' WP plugin.
